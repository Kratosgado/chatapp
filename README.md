# Demo Chat App (Spring Boot + Vue) For Learning Purposes

## Backend features

- User registration & login (JWT-based)

- Create chat rooms / private chats

- Send/receive messages

- Persist users and messages in Postgres

- Basic role-based authorization (maybe admin vs regular users)

## Frontend features (Vue)

- Login/Register pages

- List of chat rooms / users

- Chat window

- JWT stored and attached to requests

- Protected routes for logged-in users
